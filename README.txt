This plugin enables you integrate login/registration forms into custom pages, instead of using the default wordpress ones

== Description ==

Wordpress login and registration pages feels like they are not a part of your website, this plugin allows you to integrate wordpress login / registration into custom pages.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. download the plugin 
2. unzip it
3. upload it to '/wp-content/plugins/' directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Go to plugin options page to customize it.
6. Go to widgets and add the widget to your sidebar
6. that's it.
